My mind map 설명서

- 기본 동작
 : 화면에서 마우스 오른클릭하면 박스 생성
 : 박스에서부터 오른클릭으로 빈곳으로 드래그하면 파생박스 생성
 : 박스를 왼쪽 클릭으로 선택 (빨갛게 변함)
 : 빈곳을 클릭하면 선택 해제
 : 스페이스바 를 누른 상태에서 왼쪽드래그로 이동/ 빈곳일 경우 화면이동

- File 메뉴
 : new, open, save, save as, exit 

- Edit 메뉴
 : undo/redo
 : size = 박스를 먼저 마우스왼쪽클릭으로 선택해서 빨갛게 만든 후에 이 메뉴를 선택(혹은 단축키 누름)
 : delete = 박스먼저 선택, 메뉴 클릭 = 박스를 삭제 (딸린 박스들이 있는 경우 함께 삭제)
 : detach = 박스선택>메뉴클릭 = 박스를 부모로부터 독립시킴 (딸린것들이 있을경우 같이)
 : adjust = 박스선택>메뉴클릭 = 박스 크기를 내용에 맞게 자동조정

- View 메뉴
 : color = 박스 선택>메뉴클릭 = 박스 색 변경
